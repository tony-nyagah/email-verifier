Email Verifier

Verify email addresses instantly without sending any emails. Check validity, deliverability, and more.

What We Check

Syntax Validation

Validates email format and structure according to RFC standards.

Domain & MX Records

Checks if the domain exists and has valid mail exchange records.

SMTP Verification

Connects to mail server to verify if the mailbox actually exists.

Disposable Detection

Identifies temporary and disposable email addresses.

Role Account Check

Detects role-based emails like admin@, support@, info@.

Free Provider

Identifies if the email is from a free provider like Gmail, Yahoo.

API Usage

You can also use our API programmatically. Send a POST request to verify emails: 

curl -X POST http://localhost:8080/api/verify \
  -H "Content-Type: application/json" \
  -d '{"email": "user@example.com"}'
